## 🔮 O Cenário

O computador pensou em um número de **1 a 100**. Ele não vai te contar qual é, mas ele é um robô honesto: a cada tentativa sua, ele vai dizer se o número secreto é **maior** ou **menor** do que o seu palpite. 

> **Seu objetivo:** Reduzir as possibilidades matematicamente (ou na base do puro instinto) e acertar o número com a menor quantidade de jogadas possível.
